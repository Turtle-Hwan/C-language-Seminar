## 세미나 요약
> 1강 : C 언어 개요 및 작동 방식, 개발환경 구축(Visual Studio 2019)   
> 2강 : 자료형, 변수의 이해와 입출력 함수   
> 3강 : 연산자   
> 4강 : 제어문   
> 5강 : 제어문 심화    
> 6강 : 배열   
> 7강 : 포인터, 배열에서 포인터 활용   
> 8강 : 함수   
> 9강 : 문자열   
> 10강 : 구조체   
> 11강 : 변수의 생존, 데이터 세그먼트, void main     
> 12강 : 동적 메모리 할당   
> 13강 : 파일 입출력   
> 14강 : git, github 이용, VS2019와 연동, 링크드 리스트   
> 15강 : 매크로 함수, 인라인 함수, 링크드 리스트
